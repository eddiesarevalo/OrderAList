Order A List

Hello! Welcome to order a list. This application will take a list of items and order it into ascending alphabetical order. I often have to do this so I created a script to do it for me! You are welcome to use it too :)

Instructions:

  1. Check the list you wish to order. There must be one entry per line.
  2. Select the list, copy, and paste into the input box below.
  3. Press the "Order!" button
